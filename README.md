# Hermes Skills

Reusable Hermes Agent skills.

## Included

- `polymarket-sports-advisor-development` — build read-only Polymarket sports advisor tools with CLOB pricing, paper trading, CLV tracking, model calibration, and scheduled reporting.

## Install

```bash
hermes skills tap add KeaneYan/polymarket-sports-advisor-skill
hermes skills install polymarket-sports-advisor-development
```

Or inspect with the cross-agent Skills CLI:

```bash
npx --yes skills add KeaneYan/polymarket-sports-advisor-skill --list
```
